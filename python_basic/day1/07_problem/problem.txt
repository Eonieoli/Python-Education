📌 기초 문제 (15분)
1. 두 숫자를 input()으로 입력받아 사칙연산 결과 출력
2. 이름과 나이 입력받아 f-string으로 자기소개 출력

4. 문자열 거꾸로 출력하기 ([::-1])

5. 여러 줄 문자열(""")로 영수증 만들기

7. CSV 형식 문자열 파싱 ("이름,나이,이메일")
8. 입력받은 문장의 단어 개수 세기 (split 활용)

📌 도전 문제 (15분)
9. 온도 변환기 (섭씨 ↔ 화씨)
